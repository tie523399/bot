## 🚀 快速開始

1. 安裝依賴：
```bash
pip install -r requirements.txt
```

2. 設置配置：
- 編輯 `config.py`
- 設置 `API_TOKEN` 和 `ADMIN_IDS`

3. 初始化資料庫：
```bash
python init_db.py
```

4. 啟動機器人：
```bash
python main.py
```
